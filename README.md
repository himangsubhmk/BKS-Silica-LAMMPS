# BKS-Silica-LAMMPS
This is the potential file for LAMMPS for the BKS Silica potential. 
the LAMMPS potential file to simulate BKS Silica is
pair_vspbks.cpp

Parameter are kept same as
I. Saika-Voivod, F. Sciortino, and P. H. Poole, Phys. Rev. E 69, 041503 (2004)

Compilation with LAMMPS version lammps-9Dec14.

See the make.sh file for details.

A file in.nvt.BKS for NVT silumation is given for MD run

See the lammps documentation for other details.
